# Student Record Management System

 Description
This is a simple C program to manage student records using file handling. It allows users to store and retrieve student data efficiently.

 Features
- Add new student details
- View all student records
- Search student by ID
- Data stored in a file

 Technologies Used
- C Programming Language
- File Handling (fopen, fprintf, fscanf)

- Project Structure
- student_management.c → Main program file
- students.txt → Stores student data (created automatically)

Purpose
This project is created for learning basic C programming concepts like:
- Structures
- Functions
- File handling
